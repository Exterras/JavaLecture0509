객체지향

객체 : 데이터와 로직이 모두 들어있는 완전체 (필드, 메소드)

객체지향 프로그래밍 데이터 처리 방법
	
	색상이 결정되지 않은 자동차 : 클래스
	[객체생성 : 클래스 복사]
		색상이 결정된 자동차 : 객체
		
객체자료형과 변수와의 관계

	객체변수
	[ClassName className1 = new ClassName();]
		className1의 메모리 주소값이 저장된다. -< 참조형 데이터 저장방식과 똑같다.
		
객체자료형과 데이터의 관계

	객체는 클래스로부터 생성되지만 각자 생성된 객체는 완전한 독립체
	각자 생성된 객체의 데이터는 별개의 데이터이다.
	
개별 데이터가 포함 -> 클래스의 인스턴스 변수로 관리	
데이터를 업데이트 -> 변경 -> 메서드
