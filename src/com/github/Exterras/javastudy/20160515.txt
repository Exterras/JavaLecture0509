배열의 크기는 최초에 한번 실행하며 변경이 불가능하다
배열 주머니는 참조형 변수와 데이터 저장방식이 같다.
(배열의 값은 주소값으로 저장이 되며 
각 인덱스의 값은 데이터형으로 저장된다.)
int[] arr1 = {1,2,3,4,5}	// arr1 : I@15db9742 (주소값)
int[] arr2 = arr1		// arr2 : arr1(I@15db9742)
// arr1과 arr2는 같은 주소를 참조한다.
// arr2만 데이터를 변경해도 arr1의 값도 같이 변경된다.
// 참조형 변수의 특징 (레퍼런스)

다차원 배열 : 배열이 가리키는 데이터에 또 다른 배열이 들어있는 구조
(배열 안에 또 배열 넣음 -> 
배열 주소값에 또 다른 주소값들이 들어있다.)
